# DOCUMENTAZIONE_PERAGENT

Questa cartella raccoglie riferimenti rapidi per gli agent in base alle macro-aree del progetto:

- `backend/` – API FastAPI, servizi RAG, gestione corsi/libri, annotazioni, integrazioni esterne.
- `frontend/` – Next.js app, lettore PDF, chat, componenti UI condivisi.
- `ai/` – Modelli LLM supportati, orchestrazione RAG+LLM, strategie di prompting.
- `infrastructure/` – Setup locale, Docker, networking, variabili d'ambiente.
- `product/` – Flussi UX, roadmap funzionali, guide per tutor/studio.

Ogni sotto-cartella contiene dei file `*.md` con collegamenti ai documenti ufficiali già presenti nel repository (es. `docs/`, `AGENTS.md`, ecc.) così che gli agent sappiano dove trovare i dettagli completi.
